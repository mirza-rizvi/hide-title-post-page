This plugin allows authors to hide the title tag on single pages and posts via the edit post screen.

== Description ==
The Hide Title Post Page plugin adds a lightweight meta box to posts and pages that lets you toggle the visibility of the title. Use it when creating landing pages or any content where the default WordPress title should remain hidden.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/` directory or install the plugin through the WordPress Plugins screen.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Edit a post or page and locate the **Hide Title** meta box.
4. Tick the checkbox to remove the title for that specific post or page.

== Frequently Asked Questions ==
= How do I hide the title tag on a post or page? =
To hide the title tag on a post or page, edit the post or page and look for the "Hide Title" meta box on the right side. Check the "Hide the title tag on this post or page" checkbox to hide the title tag on that post or page.

= Will hiding the title affect SEO? =
Hiding the title is purely visual. Search engines will still index the post title as usual.

= Can I hide titles by default? =
No. This plugin is intentionally lightweight and only hides titles when the checkbox is selected on each post or page.
